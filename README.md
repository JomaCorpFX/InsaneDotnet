# InsaneDotnet
Insane library Compatible with .NET 5 and upwards.

## Add to your project

**Package Reference**   
```
<PackageReference Include="InsaneIO.Insane" Version="1.0.1" />
```

**Dotnet CLI**   
```
dotnet add package InsaneIO.Insane --version 1.0.1
```

**Package Manager**   
```
Install-Package InsaneIO.Insane -Version 1.0.1
```
