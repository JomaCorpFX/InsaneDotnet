# InsaneDotnet
Insane library Compatible with .NET 8 and upwards.

## Add to your project

**Package Reference**   
```
<PackageReference Include="InsaneIO.Insane" Version="8.0.0" />
```

**Dotnet CLI**   
```
dotnet add package InsaneIO.Insane --version 8.0.0
```

**Package Manager**   
```
Install-Package InsaneIO.Insane -Version 8.0.0
```
